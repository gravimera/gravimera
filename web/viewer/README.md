# Gravimera Web Viewer (MVP)

This is a minimal web viewer that:

- decodes `scene.grav` and `terrain.grav` (protobuf)
- renders terrain as a plane and objects as basic geometry primitives

## Controls

- Mouse: drag to orbit, scroll to zoom.
- Keyboard:
  - Pan: `W/A/S/D` or `Arrow` keys
  - Rotate: `Q/E` (yaw), `Z/X` (pitch)
  - Hold `Shift` to move/rotate faster
- Mobile:
  - Use the on-screen move pad (bottom-left) to move forward/back/left/right.
  - Use the "Hide Panel" button to collapse the file picker UI and get more screen space.

## Run

Prereqs:

- Node.js (npm)

From repo root:

```bash
cd web/viewer
npm install
npm run dev
```

The viewer will auto-load the default demo scene from:

- `assets/scene_wasteland/scene.grav`
- `assets/scene_wasteland/terrain.grav`

You can also load custom files from your local disk:

- `<root_dir>/realm/<realm_id>/scenes/<scene_id>/build/scene.grav`
- `<root_dir>/realm/<realm_id>/scenes/<scene_id>/build/terrain.grav` (optional)

## Run (Server)

To bind to all addresses (LAN) and pick a port:

```bash
python3 tools/run_web_viewer.py --port 5173
```

Notes:

- The proto schemas live under [`proto/`](/Users/flow/workspace/github/gravimera/proto).
- The TS decoders are generated by `ts/gravimera_proto` (see [`docs/file_formats/scene_grav.md`](/Users/flow/workspace/github/gravimera/docs/file_formats/scene_grav.md) and [`docs/file_formats/terrain_grav.md`](/Users/flow/workspace/github/gravimera/docs/file_formats/terrain_grav.md)).
  - `protoc` is only needed if you modify `.proto` files and want to regenerate the TypeScript sources via `npm run regen` in `ts/gravimera_proto`.
