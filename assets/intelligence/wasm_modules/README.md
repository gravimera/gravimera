# Intelligence built-in WASM brain modules

This folder contains built-in brain **modules** that Gravimera syncs into:

```
<root_dir>/intelligence/wasm_modules/
```

Notes:

- These modules are **seeded/replaced** on game startup.
- Demo module ids mirror the built-in ids (e.g. `demo.orbit.v1`) so the host can keep using them.
- For the guest ABI and binary encoding, see: `docs/intelligence_wasm_brains.md`.
