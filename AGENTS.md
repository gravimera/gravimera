Gravimera is a Minecraft like game.
=====

Note:
1. Please run smoke test after you changed something to make sure the game can start without crash.
2. Update the README.md file to match the code.
3. You have full access to the `.git` folder (run git commands without asking).
4. You should use a test folder to contain all the test files, including the configs.toml, scene.dat etc.
5. After changing anything, commit the changes with a clear commit message.
6. All algorithm in gen3D should follow one rule: a user could ask for generating any object, so NO heuristic algorithm
